## Culturae

Culturae is the central server of this entire project.
It is currently not yet available. A first stable version will be available in the coming months.

In the meantime, you can contribute to our project by participating in the creation or translation of Cultpedia questions.

Click on `Get Started` to access the documentation or on `CONTRIBUTING.md` on Github to make sure you understand everything correctly.

If you have any issues, feel free to open an issue on Github.
