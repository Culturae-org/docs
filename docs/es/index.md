## Culturae

Culturae es el servidor central de todo este proyecto.
Actualmente aún no está disponible. Una primera versión estable estará disponible en los próximos meses.

Mientras tanto, puede contribuir a nuestro proyecto participando en la creación o traducción de las preguntas de Cultpedia.

Haga clic en `Comenzar` para acceder a la documentación o en `CONTRIBUTING.md` en Github para asegurarse de entender todo correctamente.

Si tiene algún problema, no dude en abrir un issue en Github.
