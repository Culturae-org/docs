## Culturae

Culturae est le serveur central de tout ce projet.
Il n'est actuellement pas encore disponible. Une première version stable sera disponible dans les prochains mois.

En attendant, vous pouvez contribuer à notre projet en participant à la création ou à la traduction des questions de Cultpedia.

Cliquez sur `Commencer` pour accéder à la documentation ou sur `CONTRIBUTING.md` sur Github pour être sur de bien tout comprendre.

En cas de problème, n'hésitez pas à ouvrir une issue sur Github.
